## Gaze Estimation

The repo consists of the sample for gaze estimation solution. 

### Prerequistes

1. python <= 3.8.*
2. ubuntu machine


### Steps:
1. Clone the repo
2. create a virtual env using `python3 -m venv env`
3. activte venv `source env/bin/activate`
4. run `./setup.sh`
5. run `./demo.sh`
